# NextHire

AI-powered technical mock interview platform with advanced evaluation features.

## Features

- Technical Mock Interview Mode (Text-Based)
- Experience-Aware Evaluation Logic
- Job-Role-Aware Evaluation Logic
- Answer Gap Intelligence (AGI)
- Cognitive Latency Tolerance (CLT)
- Interview Robustness Index (IRI)
- Terminology Alignment Index (TAI)
- Answer Compression Efficiency (ACE)
- Expectation Drift Detection (EDD)

## Tech Stack

- React
- Vite
- Tailwind CSS (removed - using custom CSS)
- Spline 3D

## Getting Started

```bash
npm install
npm run dev
```
