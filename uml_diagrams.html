State Machine Diagram

Behavioral 
stateDiagram-v2
    [*] --> Idle

    Idle --> Authenticating : User opens app
    Authenticating --> Authenticated : Login success
    Authenticating --> Idle : Login failed

    Authenticated --> ResumeUploading : Upload resume
    ResumeUploading --> ResumeParsing : File stored
    ResumeParsing --> QuestionGeneration : Text extracted

    QuestionGeneration --> QuestionsReady : Gemini API returns questions
    QuestionGeneration --> GenerationFailed : API error / timeout
    GenerationFailed --> QuestionGeneration : Retry with fallback model

    QuestionsReady --> InterviewInProgress : Start interview

    state InterviewInProgress {
        [*] --> AskingQuestion
        AskingQuestion --> RecordingAnswer : AI video plays question
        RecordingAnswer --> UploadingVideo : Candidate finishes answer
        UploadingVideo --> TranscribingAnswer : Video uploaded
        TranscribingAnswer --> AskingQuestion : Next question
        TranscribingAnswer --> AllAnswered : No more questions
    }

    InterviewInProgress --> Evaluating : All questions answered
    AllAnswered --> Evaluating

    Evaluating --> FeedbackReady : AI scores and generates feedback
    FeedbackReady --> InterviewInProgress : Retry interview
    FeedbackReady --> SessionComplete : User exits

    SessionComplete --> Authenticated : Start new session
    SessionComplete --> [*] : Logout
